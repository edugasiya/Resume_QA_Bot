# Resume_QA_Bot - LLMs, LangChain, LlamaIndex

Welcome to Day 1 of the GenAI Bootcamp! This repository covers foundational concepts and hands-on exercises for building with LangChain and LlamaIndex.

## Topics Covered:
- Introduction to LLMs (Tokenization, Attention, In-context learning)
- LangChain basics with OpenAI
- LlamaIndex for document querying
- Mini project: Resume Q&A bot

## Understanding Temperature
Range: Typically from **0.0 to 1.0**.
* **Low Temperature (0.0 to 0.3):** Deterministic, focused, predictable. Ideal for factual answers, summaries, code generation.
* **High Temperature (0.7 to 1.0):** Creative, diverse. Ideal for storytelling, brainstorming.

## Setup Instructions
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add your OpenAI API Key: `OPENAI_API_KEY=sk-...`
4. Run the code: `python app.py`
